title Xóa Loại hình cho thuê
actor Admin
boundary Trang Quản lý Loại hình cho thuê
control AccomCategoryController
participant AccomCategoryService
database Database

activate Admin
Admin->Trang Quản lý Loại hình cho thuê: Truy cập trang quản lý loại hình cho thuê
activate Trang Quản lý Loại hình cho thuê
Admin->Trang Quản lý Loại hình cho thuê: Chọn loại hình cần xóa

Trang Quản lý Loại hình cho thuê->Trang Quản lý Loại hình cho thuê: Hiển thị xác nhận xóa
activate Trang Quản lý Loại hình cho thuê
Admin->Trang Quản lý Loại hình cho thuê: Nhấn nút "Xóa"

Trang Quản lý Loại hình cho thuê->AccomCategoryController: Gửi yêu cầu xóa
deactivate Trang Quản lý Loại hình cho thuê
activate AccomCategoryController
AccomCategoryController->AccomCategoryService: Xử lý yêu cầu xóa
activate AccomCategoryService
AccomCategoryService->Database:Cập nhật trạng thái deleted
activate Database
Database-->AccomCategoryService: Xác nhận đã xóa thành công
deactivate Database
AccomCategoryService-->AccomCategoryController: Trả về kết quả xóa
deactivate AccomCategoryService
AccomCategoryController-->Trang Quản lý Loại hình cho thuê: Hiển thị thông báo xóa thành công và cập nhật danh sách
deactivate AccomCategoryController
Trang Quản lý Loại hình cho thuê-->Admin: Hiển thị thông báo và danh sách loại hình đã được cập nhật
deactivate Trang Quản lý Loại hình cho thuê
deactivate Admin

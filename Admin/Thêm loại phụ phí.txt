title Thêm mới Loại phụ phí
actor Admin
boundary Trang Quản lý Loại phụ phí
control SurchargeTypeController
participant SurchargeTypeService
database Database

activate Admin
Admin->Trang Quản lý Loại phụ phí: Truy cập trang quản lý loại phụ phí
activate Trang Quản lý Loại phụ phí
Admin->Trang Quản lý Loại phụ phí: Nhấn nút "Thêm mới"

Trang Quản lý Loại phụ phí->Trang Quản lý Loại phụ phí: Mở popup thêm mới
activate Trang Quản lý Loại phụ phí
Admin->Trang Quản lý Loại phụ phí: Nhập thông tin mới và nhấn "Thêm mới"

Trang Quản lý Loại phụ phí->SurchargeTypeController: Gửi yêu cầu thêm mới
deactivate Trang Quản lý Loại phụ phí
activate SurchargeTypeController
SurchargeTypeController->SurchargeTypeService: Xử lý yêu cầu thêm mới
activate SurchargeTypeService
SurchargeTypeService->Database: Lưu dữ liệu mới vào Database
activate Database
Database-->SurchargeTypeService: Xác nhận đã thêm mới thành công
deactivate Database
SurchargeTypeService-->SurchargeTypeController: Trả về kết quả thêm mới
deactivate SurchargeTypeService
SurchargeTypeController-->Trang Quản lý Loại phụ phí: Hiển thị thông báo thành công và cập nhật danh sách
deactivate SurchargeTypeController
Trang Quản lý Loại phụ phí-->Admin: Hiển thị thông báo và danh sách loại phụ phí đã được cập nhật
deactivate Trang Quản lý Loại phụ phí
deactivate Admin

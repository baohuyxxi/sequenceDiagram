title Xem Danh sách Chỗ ở (Admin)
actor Admin
boundary Trang Quản lý Chỗ ở
control AdminController
participant AdminService
database Database

activate Admin
Admin->Trang Quản lý Chỗ ở: Truy cập trang quản lý chỗ ở
activate Trang Quản lý Chỗ ở
Trang Quản lý Chỗ ở->AdminController: Yêu cầu danh sách chỗ ở
activate AdminController
AdminController->AdminService: Gửi yêu cầu lấy danh sách chỗ ở
activate AdminService
AdminService->Database: Lấy dữ liệu chỗ ở từ Database
activate Database
Database-->AdminService: Trả về kết quả danh sách chỗ ở
deactivate Database
AdminService-->AdminController: Trả về danh sách chỗ ở
deactivate AdminService
AdminController-->Trang Quản lý Chỗ ở: Hiển thị danh sách chỗ ở
deactivate AdminController
Trang Quản lý Chỗ ở-->Admin: Hiển thị danh sách chỗ ở
deactivate Trang Quản lý Chỗ ở
deactivate Admin
